# CLAUDE.md — Инструкции для Claude по проекту LaunchFlow

## Что это за проект

Продающий лендинг для SaaS-платформы **LaunchFlow** — сервиса, который помогает экспертам самостоятельно запускать продающие вебинары и эфиры без продюсера, дизайнера и технического специалиста.

**Создатель продукта:** Лиханова Олеся — эксперт по системным запускам, основатель LaunchFlow.

---

## О пользователе

- Не имеет технического фона, учится создавать проекты
- Полностью доверяет Claude в технических решениях
- Нужны объяснения: всегда объяснять ЧТО делается и ПОЧЕМУ
- После изменений кратко пояснять, что можно безопасно менять дальше

---

## Стек

- **React 18** + **Vite 6** + **Tailwind CSS 3**
- **Lucide React** — иконки
- Node.js установлен по пути `C:\Program Files\nodejs\`

## Команды

```bash
# Запуск локального сервера (открывает браузер автоматически)
npm run dev
# → http://127.0.0.1:3000 (компьютер)
# → http://192.168.0.134:3000 (телефон в той же Wi-Fi сети)

# Сборка для публикации
npm run build
# → создаёт папку dist/ — её перетаскивают на netlify.com/drop
```

**Для запуска без терминала:** двойной клик на `ЗАПУСТИТЬ САЙТ.bat`

## Публикация

Сайт публикуется на **Netlify** методом drag & drop:
1. `npm run build` → появляется папка `dist/`
2. Перетащить `dist/` на `netlify.com/drop`
3. Ссылка остаётся той же при повторных загрузках (если залогинена)

## Telegram Mini App (tg-app)

Отдельное мини-приложение для открытия внутри Telegram.
Файлы: `public/tg-app/` → после `npm run build` попадают в `dist/tg-app/`

```
public/tg-app/
├── index.html  — точка входа, подключает SDK и файлы
├── style.css   — стили, тема через CSS-переменные Telegram
└── app.js      — навигация между экранами, отправка формы
```

**Экраны (порядок):**
1. `welcome` — приветствие, статистика, кнопка «Попробовать бесплатно»
2. `niche` — выбор ниши (6 вариантов)
3. `situation` — текущая ситуация (4 варианта)
4. `contact` — форма: имя + Telegram/телефон
5. `success` — подтверждение + кнопка «Написать в Telegram»

**Где менять данные:**
| Что | Файл | Где искать |
|---|---|---|
| Варианты ниш | `index.html` | `id="niche-options"` |
| Варианты ситуаций | `index.html` | `id="situation-options"` |
| Ссылка на Telegram-бот | `index.html` | `href="https://t.me/..."` |
| Статистика на welcome | `index.html` | `.stats-row` |
| Тема/цвета | `style.css` | `:root` переменные |

**URL после публикации:** `https://super-eclair-93d5f3.netlify.app/tg-app/`

**SDK:** Telegram Web App JS автоматически задаёт цвета через CSS-переменные (`--tg-theme-bg-color` и др.), поддерживает светлую и тёмную тему.
